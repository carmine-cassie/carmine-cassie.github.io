What I Did in March

Osawi Typeset

Around the start of this month jan [ken.][taso.mi:] put out an unabridged toki pona translation of The Wonderful Wizard of Oz, and I very quickly got very excited by the idea of reading it in physical form.

To this end, I hastily put together a typeset of it using Typst! I got rid of the illustrations, to save myself time, money, and spoilers, added some sitelen pona drop caps, which I've always loved the idea of, and sent it off to Lulu so I could get a copy printed!

It's not a super visual update this month, so enjoy these few photos!

It turned out pretty okay, though with some (huge) flaws. It's been very lovely to slowly read through it! You can find my source files here, if you like. (You'll need to view a version from before around March 10, as I've been messing with them since.)

Self Hosting Adventures

After setting up a pi-hole for my household last month, I wanted to try out some more self-hosting projects! I transfered the pi-hole onto an old laptop (with a very janky install process featuring a USB to SATA cable), and set up two additional projects:

Navidrome

I thought it would be fun to try out a music server, to potentially move away from Spotify (I keep hearing that Spotify is very evil lol). To me, it seems an awesome idea to replace a music streaming subscription fee with purchases on Bandcamp Fridays (at the same approximate rate of spending).

With that in mind, I set up a Navidrome server, which was super easy, as well a mesh VPN (for free!) with Tailscale, so I can access it from anywhere. On my phone, I access the server with Tempus - my first time installing an Android app from not-the-play-store, which was surprisingly easy!

Chord Trainer

Ever since I was a kid, each time I've played piano (or other instruments), I've always focused on plodding through memorising sheet music, one note at a time, rather than building any connection between my head, hands and music theory.

These days, my dream piano skill level is being able to simply accompany myself as I sing, because singing is super fun! I want to be able to more easily play from lead-sheet style notation, the sort you might find on Ultimate Guitar.

To that end, I put together a quick tool that generates lists and lists of chords (or intervals!) in random order, to practice finding them quickly, and to build familiarity with them all. Since building it, I've been playing some songs from Ghost Quartet and having just an absolute blast.

Shrunk A Shirt

Ever since I snagged a sewing machine at a thrift store for $15, I've wanted to do more sewing projects, and I took a stab at one of those this month.

I have a lot of slightly baggy print t-shirts with cool designs, and a few baby tees (in this case $4 kids shirts from Big W), which fit me well, and so I've been thinking about taking in the print tees to fit more like baby tees.

I gave that idea a practice go, bought an XL shirt from a thrift store and turned it into an XS! (Following this tutorial.) There were parts where I got super frustrated, and the permanence of cutting off the excess fabric was scary, but I pulled through and now it's the best fitting shirt I own! Maybe I'll do this to more shirts in the future!
